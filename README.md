### Quickstart repo for https://github.com/noCaptchaAi/nocaptchaai-puppeteer

```bash
git clone https://github.com/noCaptchaAi/puppeteer-hcaptcha-solver
cd puppeteer-hcaptcha-solver
npm i
npm run start
```
